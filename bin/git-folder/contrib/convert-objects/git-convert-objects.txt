git-convert-objects(1)
======================

NAME
----
git-convert-objects - Converts old-style git repository


SYNOPSIS
--------
'git-convert-objects'

DESCRIPTION
-----------
Converts old-style git repository to the latest format

------
Written by Linus Torvalds <torvalds@osdl.org>
